# Dashwright Helm Charts

This repository hosts Helm charts for Dashwright.

## Usage

```bash
helm repo add dashwright https://cybedefend.github.io/dashwright
helm repo update
helm install dashwright dashwright/dashwright
```

## Available Charts

- **dashwright**: Modern Playwright dashboard for visualizing test runs

For more information, visit the [main repository](https://github.com/CybeDefend/dashwright).
